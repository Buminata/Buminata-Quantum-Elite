# Buminata Quantum Elite v7.5 📉

**Buminata Quantum Elite** is an advanced AI-powered market analytics platform for US Stocks and Cryptocurrencies. Built with a high-performance Hybrid AI Engine, it provides real-time insights and "High Reward" trading signals.

## 🚀 Live Demo

[Buminata Quantum Elite Dashboard](https://buminata-quantum-elite.streamlit.app/)

## ✨ Key Features

- **Hybrid AI Engine**: Combines **XGBoost** and **Deep Learning (MLP)** for superior price projection accuracy.
- **High-Reward Spot Advisor**: Advanced entry/exit logic with custom Risk/Reward requirements (1:2 to 1:4+).
- **Quantum Comparison Engine**: Live cross-asset correlation analysis and normalized trend comparison.
- **Deep Technical Analytics**: Automated Support/Resistance detection, Bollinger Bands, RSI, MACD, and ATR.
- **Market-Aware Logic**: Special "Next Day Outlook" mode for stocks during market closures.
- **High-Fidelity UI**: Premium TradingView-inspired dark theme with real-time pulsing price feeds.

## 🛠 Tech Stack

- **Framework**: [Streamlit](https://streamlit.io/)
- **Data Engine**: [Yahoo Finance API (yfinance)](https://github.com/ranaroussi/yfinance)
- **AI/ML**: [XGBoost](https://xgboost.readthedocs.io/), [Scikit-learn (MLP & Scalers)](https://scikit-learn.org/)
- **Visualization**: [Plotly](https://plotly.com/python/graph-objects/)

## 🚀 Quick Start

### 1. Requirements

Ensure you have Python 3.9+ installed.

### 2. Installation

```bash
pip install streamlit yfinance pandas numpy plotly xgboost scikit-learn python-dotenv streamlit-autorefresh
```

### 3. Run the Application

```bash
python -m streamlit run app_st.py
```
